1. Вступление
----------------

Мод изменяет пользовательский интерфейс в играх Baldur's Gate Siedge of Dragonspear, Baldur's Gate II Enhanced Edition и Enhanced Edition Trilogy.

Основные изменения в интерфейсе:
1.1. Лист персонажа
	Прогрессбары очков опыта теперь начинаются с начала при получении новвого уровня
	Характеристики персонажа теперь разделены на группы и каждую группы можно скрыть или отобразить. Кнопки +/- сворачивают/разворачивают все группы
	Клик по названию атрибута(н-р: Сила) выводит его описание и преимущества, которые он даёт персонажу.
	Клик по полю, содержащему название расы, класса, мировоззрения выводит описание расы персонажа.
	Описание класса ттеперь содержит полное описание. Если у персонажа есть кит, то после описания кита будет описание чистого класса. Для мультиклассов после его описания отображаются описания отдельных классов и, при необходимости, кита.
	При добавлении персонажу второго класса, при выборе Оружейных навыков серым цветом обозначаются навыки, в которых есть звезды в первом классе.

1.2. Главный игровой экран:
	1.2.1. Портреты персонажей
	При повышении уровня рядом с портретом персонажа появится кнопка с изображением знака "+". При нажатии по ней левой кнопкой мыши откроется окно повышения уровня. При нажатии правой кнопкой - Лист персонажа.
	1.2.2. Окно диалога, сообщений и панель действий.
	Изменять размер окна сообщений можно теперь не только в высоту, но и в ширину
	В окне диалогов текст, как обычно, отображается слева от портрета. Но если у персонажа нет портрета, текст теперь растягивается на всю ширину окна.
	Рядом с переключателем Быстрого сбора трофеев добавлен переключатель Авто-сбора трофеев. Это мод от bubb'а с некоторыми изменениями.

1.3. Создание персонажа
	Добавлена возможность создания Случайного персонажа.
	Выбор портрета теперь производится из списка портретов. Рядом с портретом будет отображаться название файла. И его описание, если оно есть(описание можно настроить в файле m_zZport.lua).
	Сохранение атрибутов. Теперь отображаются все значения сохраненных атрибутов, общее количество очков текущих и сохраненных атрибутов и разница между ними.
	Для следопытов кроме описание расового врага добавлено его изображение.

1.4. Инвентарь
	Добавлен общий инвентарь. В него можно попасть через специальную кнопку. Для выхода нажать кнопку Esc или кликнуть по глазу дракона в правом верхнем углу.
	В инвентарь добавлен быстрый сбор трофеев. Примечание: из-за ограничений игрового движка при сборе трофеев они будут попадать в рюкзак персонажа, для которого был открыт инвентарь. Но в  то же время это может облегчить сбор трофеев в Общем инвентаре.
	Добавлено меню идентификации. Его подробное описание можно посмотреть в игре.

1.5. Дневник
	Дневник теперь имеет два варианта отображения. Маленький, стандартный для Baldur's Gate (II) Enhanced Edition, и Большой, похожий на дневник из старых версий игры.
	При добавлении в девник новой записи, раздел, в который она добавлена, отмечается галочкой.
	Добавлено меню просмотра последних событий.
	Добавлено меню просмотра всех квестов в одном списке. Слева будет показан список квестов, справа - записи журнала, относящиеся к этому квесту, с обозначением главы, в которой она была добавлена. Двойной щелчок по записи сворачивает/разворачивает ее.
	"Важные события" теперь всегда отображаются в активных заданиях.
	Для Трилогии "Важные события" первой и второй части объединены в одну группу.

1.6. Магазины/Храмы
	При торговле предметы в инвентаре персонажа сортируются. Первыми отображаются контейнеры, за ними предметы, которые можно продать в этом магазине, и последними предметы, которые продать нельзя.
	Через специальные переключатель можно посмотреть предметы, которые надеты на персонажа не выходя из магазина.
	Есть возможность выбрать для продажи сразу все предметы.
	Есть возможность украсть сразу несколько предметов.
	Нельзя выделить для покупки предметы, на которые не хватает золота (их можно выделить в том случае, если персонаж может воровать).
	Фильтрация предметов осуществляется двумя способами. По поиску символов в названию предмета, и по типу предметов.
	В окне идентификации отображаются только те предметы, которые необходимо идентифицировать.
	В меню Пожертвования можно выбрать на сколько увеличивать/уменьшать сумму пожертвования при клике на кнопки +/-.
	В меню Лечения теперь не доступны заклинания, на которые не хватает золота.
	В меню Напитки добавлены иконки для напитков и невозможность их выделить при нехватке золота.

1.7. Книга заклинаний/Свиток священника
	Теперь имеет два варианта отображения. Стандартный для Baldur's Gate (II) Enhanced Edition, и второй, больше похожий на оригинальные игры.
	Переключать внешний вид можно в меню Настройки или кликнуть по глазу дракона в правом верхнем углу.

1.8. Карта мира
	Клик по кнопке со стрелками справа центрирует экран на текущей локации.

1.9. Эпилоги
	Над портретом персонажа отображается его имя.

1.10. Сохранение/Загрузка
	Добавлена фильтрация по имени сохранения, имени персонажа и по главе.

1.11. Консоль
	В список последних команд теперь не добавляются повторяющиеся команды.
	Двойной клик по команде в списке сразу же выполняет ее. Правый клик - удаляет из списка.

1.12. Меню загрузки игры
	Все кнопки теперь доступны на одном экране и теперь не будет переключений между экранами. На этом же экране можно выбрать игру.
	В меню Настройки наа этом экране можно посмотреть список присоединяемых персонажей(включая персонажей из модов) с их характеристиками и биографией.

1.13. Достижения
	На левой панели главного игрового экрана добавлена кнопка "Достижения". Ее нажатие открывает окно Достижений.
	Так же при выполнении достижения отображается всплывающее окно, как и с записями дневника.

Примечания: многие новые возможности по умолчанию отключены. Включить их можно в меню настроек.


----------------
2. Установка
----------------
Windows:
--------

Это WeiDU-мод, что значит, что его очень легко установить. Просто распакуйте загруженный архив в каталог с вашей игрой (где расположен файл "chitin.key") и запустите "setup-tipun_gui.exe". Следуйте инструкциям.

Чтобы удалить мод, запустите "setup-tipun_gui.exe" и снова следуйте инструкциям.

----------------
2.1. Порядок установки
----------------
Мод содержит два компонента.
Первый компонент: Tipun's User Interface (main component).
Его лучше устанавливать в самом начале установки. После EET при установка на Трилогию.
Второй компонент: Tipun's User Interface (create data tables)
Его нужно устанавливать в том случае, если вы устанвили другие моды после первого компонента(это касается модов добавляющих предметы, киты, НПС). Этот компонент нужно ставить в самом конце установки.

5. История версий
-------------------
Версия v2.0
- при покупке/продаже добавлена возможность посмотреть количество очков в оружейных навыках
- исправлена ошибка во время идентификации предметов в магазинах при смене персонажа
- убрано растягивание текста на всю ширину окна в окне диалогов при отсутствии портрета у собеседника(из-за проблемы "схлопывания" текста, которая пока не решена)
- добавлены индикаторы повышения уровня слева от портретов на главном игровом экране (включается в настройках)
- некоторые небольшие исправления
Второй компонент:
- исправление положения элементов интерфейса SCS и Hidden Gameplay Options в меню Настроек
- если установлен мод Infinity Colors, то индикаторы повышения уровня возле портретов будут того же цвета, что и рамка портретов (только если не менялся определенный фрагмент в файле Baldur.exe)
Прблемы:
- если загрузить сохранение, сделанное до установки мода и у кого-то из персонажей можно повысить уровень, иногда индикаторы опыта отображаются неправильно. После повышения уровня все стабилизируется. Пока не нашел причину того, почему так происходит.

- добавлены файлы ридми
- перекрашена кнопка Достижений
- исправлена кнопка "Об Авторах" в SoD
